Channels
  1. Add bot to your channel as admin
  2. Use /translate command for specific translations
  3. Use /detect to identify message language
  4. Enable auto-translation for the channel

Setup Instructions

Getting Started
  1. Get Bot Token: Message @BotFather on Telegram
  2. Create Bot: Use /newbot command and follow instructions
  3. Configure: Add your bot token to the .env file
  4. Run Bot: Execute python telegram_bot.py or python run_bot.py
  5. Start Chatting: Find your bot on Telegram and start translating!
For Channels
  1. Add your bot to the channel as an administrator
  2. Use python channel_bot.py for channel-specific features
  3. Get channel ID by forwarding a message to @userinfobot
